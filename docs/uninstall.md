# Uninstallation

## Windows

1. Open **Settings → Apps** (or **Control Panel → Programs → Uninstall a Program**).
2. Find **Chanterelle** in the list.
3. Click **Uninstall** and follow the prompts.

### Removing app data

Uninstalling the app does **not** delete your projects directory or any project files — those are wherever you chose to keep them and are fully yours.

If you also want to remove Chanterelle's stored settings (projects directory path, theme preference, etc.), delete the app data folder:

```
%APPDATA%\com.chanterelle.app\
```

To open this path, press **Win + R**, type `%APPDATA%`, and press Enter. Then delete the `com.chanterelle.app` folder.
